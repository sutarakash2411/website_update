welcome to modified Website Update Project

This webpage has been updated as part of the Website Update Project using Jenkins and Ansible automation.

Steps to update this page:

  • A custom index.html file was created and pushed to a GitHub repository.
  • Jenkins pulled the updated file from the GitHub repository using a Jenkins job.
  • Ansible was triggered within the same Jenkins job to copy the updated index.html file to the Apache webserver on a remote EC2 instance.
  • The Apache webserver was restarted to apply the changes and display this updated webpage.
